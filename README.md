# ApexcifyTechnologys Rock Paper Scissors Game

This is a simple, interactive **Rock Paper Scissors** game where you play against an AI opponent. It's a classic hand game brought to the command line, built with Python.

## 🚀 About the Project

This project provides a fun and quick way to play Rock Paper Scissors against a computer. The core logic focuses on taking player input, generating a random choice for the AI, determining the winner based on standard rules, and displaying the result.

## 🛠️ Built With

*   **Language:** [Python](https://www.python.org/) (Version 3.x recommended)
*   **Core Concepts:** Random module, conditional logic (if/else), user input handling.

## 📋 Prerequisites

Before you can run the game, you'll need to have Python installed on your system.

*   **Python:** The game is written in Python. You can download it from the [official Python website](https://www.python.org/downloads/).
*   **Check Installation:** To verify Python is installed, open your terminal or command prompt and run:
    ```bash
    python --version
    ```
    or for some systems:
    ```bash
    python3 --version
    ```

## ⚙️ Installation and Setup

Getting the game running on your local machine is straightforward.

1.  **Clone the repository**
    Open your terminal and use the following command to download the project:
    ```bash
    git clone https://github.com/alibutt2882/ApexcifyTechnologys_Rock_Paper_Scissors-game.git
    ```

2.  **Navigate to the project directory**
    ```bash
    cd ApexcifyTechnologys_Rock_Paper_Scissors-game
    ```

## 🎮 How to Play the Game

The game is run directly from the terminal.

1.  **Run the Python script**
    Execute the main Python file using the command:
    ```bash
    python main.py
    ```
    (You might need to use `python3 main.py` on some systems like macOS or Linux).

2.  **Gameplay Instructions**
    *   Once the script starts, the game will prompt you to make a choice.
    *   Typically, you will be asked to type `rock`, `paper`, or `scissors` and press Enter.
    *   The AI will then randomly select its choice.
    *   The game will announce both choices and declare the winner (You win, AI wins, or it's a tie).
    *   After each round, the game might ask if you want to play again. Follow the on-screen prompts (e.g., type 'yes' or 'no').
